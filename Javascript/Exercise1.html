    3. What are the differences between `==` and `===`? Write an example for each case (if any)?

    • `==` Comparing the value. VD: true == 1; (true) "2" == 2; (true)
    • `===` This means both the type and the value comparing have to be the same. VD: true === 1 (false); "2" === 2 (false); 2===2 (true);
